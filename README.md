## Prerequisites

- git
- python 3.10+
- pip
- virtualenv

## Install

1. Clone repo

```
git clone https://github.com/jeremiahvuong/jikan-rec-mapper.git
```

2. CD into repo

```
cd jikan-rec-mapper
```

3. Create a venv in your directory.

```
python3 -m venv venv
```

4. Activate venv

5. Install dependencies.

```
pip3 install -r requirements.txt
```

6. Run flask app

```
python3 main.py
```
